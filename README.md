## 简介

- 荣耀AI游戏管家安装工具，用于在尚未完成适配的荣耀笔记本设备上，尝鲜体验荣耀AI游戏管家。

## 功能

1. 静默安装荣耀AI游戏管家
2. 静默卸载荣耀AI游戏管家
3. 服务状态检查

## 注意
- *请确保设备中的[PowerShell](https://learn.microsoft.com/zh-tw/powershell/scripting/install/installing-powershell-on-windows)能够正常工作!否则可能导致程序运行出现错误!*
